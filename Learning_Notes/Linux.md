# Linux Basic Practice
리눅스 기초 공부 레포지토리

## Basic Command
| 명령어 | 의미 | 사용 방법 |
| --- | --- | --- |
| `pwd` | 현재 위치 확인 | `pwd` |
| `ls` | 현재 폴더 안의 파일/폴더 목록 보기 | `ls` |
| `cd` | 다른 폴더로 이동 | `cd 폴더명` |
| `mkdir` | 새 폴더 만들기 | `mkdir 폴더명` |
| `touch` | 새 파일 만들기 | `touch 파일명` |
| `cat` | 파일 전체 내용 출력 | `cat 파일명` |
| `head` | 파일 앞 부분 보기 | `head 파일명` |
| `tail` | 파일 뒷 부분 보기 | `tail 파일명` |
| `cp` | 파일 복사 | `cp 복사대상 복사될파일명` |
| `mv` | 파일 이동 or 파일명 변경 | `mv 변경대상파일 변경할이름` |
| `rm` | 파일 삭제 | `rm 파일명` |

## less
less는 긴 파일을 페이지 단위로 보는 명령어
`less 파일명`으로 사용, 아래 키로 `less`내 조작 가능
| 명령어 | 의미 |
| --- | --- |
| `Space` | 다음 페이지 |
| `b` | 이전 페이지 |
| `/검색어` | 단어 검색 |
| `q` | 종료 |

## echo
echo 터미널에 글자를 출력하는 명령어
- echo `>`: 기존 내용을 덮어쓰고 저장

| 명령어 | 의미 |
| --- | --- |
| `echo "출력 내용" > 파일` | `"출력 내용"`을 `파일`에 덮어쓰고 저장 |
| `echo "출력 내용" >> 파일` | `"출력 내용"`을 `파일` 뒤에 내용 추가 |

## Intermediate Command
| 명령어 | 의미 | 사용 방법 |
| --- | --- | --- |
| `grep` | 파일 내용에서 `특정 문자열` 검색 | grep `검색 내용` `검색 파일` |
| `find` | 경로 내 파일or디렉터리 검색, .-name  | find `위치` `기준 옵션` `확장자` |
| `wc` | 줄 수, 단어 수, 글자 수 세기 | wc `옵션` `대상 파일` |
| `sort` | 줄 단위 정렬 | sort `대상 파일` |
| `sort \| uniq` | 중복 없는 줄 단위 정렬 | sort `대상 파일` `\| uniq `|

### find 사용 예시
- find . -name "*.txt" → 현 위치(.)서부터 확장자가 txt 파일 검색
### wc 옵션 정리
| 옵션 | 의미 | 예시 |
| --- | --- | --- |
| `-l` | 줄 수 세기 | wc -l app.log |
| `-w` | 단어 수 세기 | wc -w app.log |
| `-c` | 바이트 수 세기 |	wc -c app.log |
| `-m` | 문자 수 세기 |	wc -m app.log |
